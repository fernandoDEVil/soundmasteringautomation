Se sube el volumen según el estandar de las plataformas como Youtube, Nteflix  y Spotify y no en bruto. El archivo Names.txt es para guardar el nombre original del  archivo y poner uno mas corto para evitar errores en la terminal. Los archvos deben estar juntos ademas de la siguiente estructura de carpetas y el ffmpeg instalado junto con el python:

logs
outout
subtitles
temp

Para iniciar la remasterizacion arrastrar el archivo hasta el launcher.bat; luego del proceso el archivo queda en la carpeta output y los subtitulos en subtitles. Si se genera un error haccerlo de forma manual con los comandos dados en ManualCommands.txt.
